### Project 0: 2048

- **Compile files**:  
Make sure ```pwd``` is ```proj0/``` and run:  
```javac -cp "javalib/*" game2048/*.java```
- **Run tests**  
```java -cp "javalib/*:game2048:." org.junit.runner.JUnitCore game2048.TestModel```
- **Run the game in GUI:**  
```java -cp "javalib/*:game2048:." game2048.Main```
